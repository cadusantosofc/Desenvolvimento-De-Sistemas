Box Model

O Box Model é um conceito fundamental no CSS que define como os elementos HTML são representados na página. Ele determina o tamanho total de um elemento e como o conteúdo, preenchimento, borda e margem interagem. O modelo é composto por quatro camadas: 1- Content (Conteúdo) - A área onde o texto, imagens ou outros elementos são exibidos. 2- Padding (Preenchimento) - O espaço interno entre o conteúdo e a borda. 3- Border (Borda) - A linha ao redor do elemento, que pode ter diferentes estilos e espessuras.4- Margin (Margem) - O espaço externo que separa um elemento de outros elementos.
